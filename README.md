# Coffee shop - backend

## Tech stack:

- TypeScript, Nest, Prisma
- ESlint, Prettier

## Project launch:

Clone repository:

```bash
   git clone https://github.com/aanndrewss/coffee-shop-backend.git
```

Install packages:

```bash
      npm i
```

Running the app:

```bash
# development
$ npm run start
# watch mode
$ npm run start:dev
# production mode
$ npm run start:prod
```
